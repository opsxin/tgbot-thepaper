# thepaper telegram bot

获取澎湃新闻每日热点资讯。

## 使用

机器人 URL：[thepapercn_bot](https://t.me/thepapercn_bot)

## 命令

- `get_day` - 获取今日热点新闻
- `get_days` - 获取三天热点新闻
- `get_week` - 获取这周热点新闻
- `get_day_topic` - 获取今日热点话题
- `get_days_topic` - 获取三天热点话题
- `get_week_topic` - 获取这周热点话题
- `get_comment` - 获取今日热评
- `get_answer`- 获取热点问答
